YouTube Notifications

Privacy Policy

This Privacy Policy describes how "YouTube Notifications" collects and stores user's data. Data collected by this browser extension is stored locally, never externally.

Information Collected

Once the browser extension receives an authentication token from YouTube's Data API authorized by the user, the browser extension will only use that token to collect the user's subscription information from YouTube's Data API. Otherwise, the subscription information is collected from the user inputting the channels manually. We do not track or collect any anonymous tracking information from the user.

The Use of Collected Data

Any data collected by the browser extension is stored locally and is never shared or sent externally.
